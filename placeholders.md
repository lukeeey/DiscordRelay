# Placeholders
The below placeholders are exposed on the Bukkit plugin if PlaceholderAPI is installed.

| Placeholder                          | Description                                |
|--------------------------------------|--------------------------------------------|
| `discord_guild_member_count`         | Number of members in the guild.            |
| `discord_guild_id`                   | The guild's ID.                            |
| `discord_guild_name`                 | The guild's name.                          |
| `discord_guild_icon_id`              | The guild's icon ID.                       |
| `discord_guild_icon_url`             | The guild's icon URL.                      |
| `discord_guild_splash_id`            | The guild's splash ID.                     |
| `discord_guild_splash_url`           | The guild's splash URL.                    |
| `discord_guild_vanity_code`          | The guild's vanity code.                   |
| `discord_guild_vanity_url`           | The guild's vanity URL.                    |
| `discord_guild_description`          | The guild's description.                   |
| `discord_guild_banner_id`            | The guild's banner ID.                     |
| `discord_guild_banner_url`           | The guild's banner URL.                    |
| `discord_guild_boost_count`          | The guild's boost count.                   |
| `discord_guild_max_bitrate`          | The guild's maximum voice channel bitrate. |
| `discord_guild_max_emotes`           | The guild's maximum number of emotes.      |
| `discord_guild_max_members`          | The guild's maximum member count.          |
| `discord_guild_afk_channel_id`       | ID of the guild's AFK channel.             |
| `discord_guild_afk_channel_name`     | Name of the guild's AFK channel.           |
| `discord_guild_system_channel_id`    | ID of the guild's system channel.          |
| `discord_guild_system_channel_name`  | Name of the guild's system channel.        |
| `discord_guild_owner_id`             | ID of the guild owner.                     |
| `discord_guild_owner_nickname`       | Nickname of the guild owner.               |
| `discord_guild_owner_effective_name` | Effective display name of the guild owner. |
